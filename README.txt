ShootingGame1

プレイヤー操作　十字キーで移動、スペースキーで弾を発射。
StraightEnemy    まっすぐに降りてくる通常のエネミー。初期体力5。
RandomEnemy      左右ランダムに移動する敵。初期体力2。
DropEnemy        縦に加速する敵。初期体力1。
CurveEnemy　　　 プレイヤーの方向に向かって横に移動する敵。初期体力3。
EnemyBace        敵の基地。ここから敵が出現する。これを倒すとクリアになる。
EscapeEnemy      プレイヤーから逃げるエネミー。襲ってくることはないが、倒すと高スコアなのでかわいそうだけど倒そう。
